# QuickCart

A tiny static demo store — three products, a cart, and a total. No build step.

## Run locally

```bash
python3 -m http.server 8000
# open http://localhost:8000
```

Or just open `index.html` in a browser.

## Files

- `index.html` — markup
- `styles.css` — styling
- `app.js` — cart logic and rendering
